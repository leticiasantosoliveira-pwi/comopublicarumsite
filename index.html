Como publicar um site em um provedor gratuito em 10 passos!


Vamos usar como exemplo o GitHub


Vamos ao passo a passo!


1. Abra o site GitHub e faça o seu cadastro, clicando em "create an account".

2. No próximo passo você terá que informar seu e-mail, criar uma senha, um username que você escolher e depois responder se deseja ou não receber novidades do servidor.

3. Para finalizar a criação da conta é necessário que você passe por um teste para provar que você é um ser humano, e quando estiver concluido o teste clique no botão "create account", para seguir.

4. O servidor enviará um link para o e-mail que foi usado na criação da conta, clique no link para poder continuar.

5. Na tela que abrirá clique no botão "create a repository", para começar a criar o seu projeto.

6. Coloque um nome no seu projeto, deixe ele como público, selecione "add a README file" e clique no botão create repository.

7. Em seguida clique no botão "add file" e em "upload file", na próxima tela que aparecer clique em "choose your files" para escolher os arquivos do seu site. E clique em "Commit changes"

8. Depois que os arquivos foram adicionados clique em "settings", no menu.

9. Role a página até a parte que está escrito GitHub Pages e clique no link "clik it out here" para ir a página publicar o site

10. Onde está escrito "none", selecione "main" e na próxima opção selecione "/docs, quando o seu site não apresentar imagens" ou "/root, quando o seu site possuir imagens" e salve no botão "save", para gerar o link de seu site. (esse processo pode demorar um pouco).

Pronto! Seu site já foi publicado no provedor gratuito!
